# ☕ Уютная кофейня

Заказ напитков и десертов через удобное приложение.

## Что реализовано

- Вход в систему (админ и обычный пользователь).
- Меню напитков и десертов, загружаемое из базы данных SQLite.
- Выбор сиропов к напиткам (карамель, ваниль, шоколад и др.)
- Корзина с изменением количества, удалением и редактированием сиропов.
- Баланс пользователя (пополнение, оплата).
- Просмотр истории своих заказов с детализацией.
- Печать чека в текстовый файл.
- Панель администратора:
  - Управление товарами (добавить, изменить, удалить).
  - Склад ингредиентов (пополнение, списание, точное количество).
  - Просмотр и смена статусов заказов.
  - Управление пользователями (роль, баланс, удаление).
  - Статистика по заказам.

## Тестовые учётные записи

- **Админ:** `admin` / `admin`
- **Пользователь:** `user` / `user`

## Как запустить

1. Клонируйте репозиторий в Visual Studio.
2. Запустите проект (F5). База данных создастся автоматически.

> **Если возникла ошибка SQL**  
> При клонировании репозитория база данных не переносится. Если вы получили эту ошибку:
> 1. Закройте программу.
> 2. В папке проекта перейдите в `bin\Debug` и удалите файл `coffee.db` (и `coffee.db-journal`, если есть).
> 3. Запустите программу заново – база создастся правильно, и вы сможете войти.

## Примечания

- База данных `coffee.db` появится в папке `bin\Debug` после первого запуска.
- Если нужно сбросить всё до исходного состояния – просто удалите файл `coffee.db` и перезапустите программу.
